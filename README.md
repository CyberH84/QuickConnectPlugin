# Quick Connect Plugin

Quick Connect is a plugin for [KeePass](http://keepass.info) password manager that allows you to connect to Windows/Linux/ESXi hosts.

## Requirements

- Microsoft Windows XP/7 with .NET Framework 2.0;
- [KeePass](http://keepass.info) version 2.x.

The main repository is hosted on [GitHub](https://github.com/cristianst85/QuickConnectPlugin).
